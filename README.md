# Ultron
奥创（Ultron）CMS3.0  工具

## 安装

node >= 5.0
electron-built >=0.36.0
electron-builder >=2.4.0

```
npm install electron-built && electron-builder && github-markdown -g
```

## 调试

```
npm run dev

```

再打开一个控制台

```
cd app
electron .
```

## 打包

### WINDOWS

```
npm run build:vue
npm run build:win
npm run pack:win
```

### OSX

```
npm run build:vue
npm run build:osx
npm run pack:osx
```


## 更新日志

```
npm run md
```
